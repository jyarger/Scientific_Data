# Nuclear Magnetic Resonance (NMR) Data
This folder in the Scientific_Data repository has Nuclear Magnetic Resonance (NMR) Data.  The data is mostly [Bruker](https://www.bruker.com/en/products-and-solutions/mr.html) NMR data collected using Bruker's TopSpin software on their various NMR instruments. However, there can also be data from old Agilent or Varian NMR systems (VNMRj - https://openvnmrj.org/) and even some new benchtop NMR systems (e.g., [Magritek](https://magritek.com/)).
